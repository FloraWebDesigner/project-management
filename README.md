# Quirky Directory

Quirky Directory is a project management system designed for student-created quirky websites. It allows students to log in and upload their project assignments, while teachers have the ability to log in and manage both the students and their projects. The system ensures smooth project tracking and management through an intuitive interface.

## Features

- **Student Login**: Students can create accounts, log in, and upload their project assignments.
- **Teacher Login**: Teachers can log in to view, manage, and evaluate student projects.
- **Project Management**: Teachers can assign grades, approve projects, and track students' progress.
- **Project Upload**: Students can upload their quirky website projects and provide necessary details.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js (Express)
- **Database**: MongoDB
